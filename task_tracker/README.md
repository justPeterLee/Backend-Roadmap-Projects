# Task Tracker

A simple CLI application to track and manage task
This project is a solution to [task-tracker](https://roadmap.sh/projects/task-tracker) from (roadmap.sh)

## Features

- Generate JSON file to store created task.
- Create tasks with a unique ID.
- Update tasks description and status.
- Delete tasks by their ID.
- Fetch tasks based on status: `todo` , `in-progress`, or `done.

## Prerequisites

- Node.js installed on your system
